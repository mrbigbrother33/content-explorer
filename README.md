# Content Explorer

Lille read-only frontend-værktøj til at browse indhold fra et API.

## Fokus

- async/await + Fetch API
- UI states (loading/error/empty)
- DOM rendering uden frameworks
- simpel struktur og læsbar kode

## Kør lokalt

Åbn `index.html` direkte i browseren, eller start en lokal server:

```bash
python -m http.server 8080
# eller
npx serve .
```
